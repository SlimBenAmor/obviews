Obviews V1.0
OTAWA Binary Viewers

Obviews stands for OTAWA's Binary Viewers and is used to display to human user the result of Worst Case Execution Time calculation.

Usage context

Obviews needs the output from different OTAWA's application. The most used one is maybe owcet:

			$ owcet -s HARDWARE EXEC TASK --stats

HARDWARE selects the script to compute the WCET for for the executable EXEC and the task TASK. TASK is required to be a function name and, if omitted, is the main itself.

The option --stats generates several files in the directory EXECotawa/TASK the represents the program (in its different views) and statistics collected from the computation of the WCET. These views and statistics depends on the script HARDWARE.

then obviews can be launched this way:

				$ obviews.py EXEC TASK

TASK name is optional and taken to be main if omitted. This will open a new window/tab in your preferred browser where you explore the different statistics obtained from WCET calculation.
